# Rainfall-Multiple-Linear-Regression-Analysis-in-India
This project studies how rainfall is affected by climate modes (ENSO, IOD, TMI) during JJA season.
## Objective
To analyze the impact of major climate modes (ENSO, IOD, TMI) on rainfall over India during the JJA season using multiple linear regression.

## Data Used
- Rainfall data (gridded)
- ENSO index
- Indian Ocean Dipole (IOD)
- Tropical Mode Index (TMI)

## Methodology
- Seasonal averaging (JJA)
- Multiple Linear Regression using statsmodels
- Statistical significance testing (p < 0.05)
- Spatial visualization using Cartopy

## Results
- Spatial regression maps showing influence of ENSO, IOD, and TMI on rainfall
- Significant regions identified using p-values

## Tools & Libraries
- Python
- NumPy, Pandas, Xarray
- Matplotlib, Cartopy
- Statsmodels
